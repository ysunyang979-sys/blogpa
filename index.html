Windows 平台

Clash Verge Rev

主力推荐

原版 Clash Verge 的续作，基于 Rust 和 Tauri 开发。比老掉牙的 CFW 更轻量、更快速、更现代。

  • 内置 Mihomo (Clash Meta) 内核
  • 支持 Reality, Hysteria2, TUIC v5
  • 强大的 TUN 模式与脚本管理
  • 颜值天花板，支持 CSS 注入
优点缺点
界面美观，UI 顶流有一定的学习成本
内核更新快，功能全老旧系统兼容性一般
点击下载

v2rayN

全能老兵

Windows 平台上历史最悠久、用户基数最大的客户端。多内核支持，灵活切换。

  • 支持 Xray, Sing-box, Mihomo 等多内核
  • 协议支持极其广泛，更新极其勤快
  • 完善的 TUN 模式与右键菜单
  • 极其稳定的节点管理与分享
点击下载

Karing

全平台统一

基于 Flutter 开发的新一代工具。主打多端体验一致，完美融合 Clash 规则与 Sing-box 性能。

  • 恐怖的订阅兼容性，通吃所有格式
  • WebDAV 跨端同步，配置一次全家受用
  • 极简 UI，内置“新手模式”
  • 完美适配 Apple TV
点击下载

Hiddify

智能自动化

基于 Sing-box 内核的次世代客户端。目标是“让代理消失”，极其适合懒人小白。

  • 一键智能测速与最优节点选择
  • 内置地区预设规则 (China 模式)
  • 可视化流量数据统计，颜值极高
  • 全格式链接通吃
点击下载

Android 平台

v2rayNG

国民级应用

安卓端最纯粹、断连率最低的客户端。新版本 2.0 引入了策略组和 Material You 设计。

  • 极致省电，后台保活能力极强
  • 完美支持分应用代理
  • 更新神速，紧跟 Xray 内核补丁
  • UI 随壁纸自动变色 (Material 3)
点击下载

Clash Meta Android

协议绞肉机

继承 CFA 经典 UI，内置强大的 Mihomo 内核。原生支持 Reality, Hy2, TUIC v5。

  • 扫码即用，零成本接盘老版操作
  • 变态级的轻量与低功耗
  • 手术刀式的分流与路由控制
  • 极致性能，适合旧手机旁路由
点击下载

Nekobox

性能怪兽

基于 Sing-box 内核，支持链式代理，极致追求测速与性能。

  • 支持“链式代理”跳板，保护 IP 纯净
  • 协议全家桶之王，支持 xhttp 等新协议
  • 强大的配置解析器，兼容多种订阅
  • 开源透明，追求内核极致控制
点击下载

iOS 平台

Shadowrocket

统治级霸主

传说中的“小火箭”。iOS 上最出名、扫码即用的全能选手。性价比之王。

  • 真正的协议百科全书，什么都能跑
  • 扫码导入极其方便，小白首选
  • 支持模块系统 (Module) 去广告
  • 2.99 刀买断制，永久更新
美区 App Store

Stash

Clash 苹果版

iOS 上第一款深度兼容 Clash 配置的顶级客户端。内嵌 Mihomo 内核，重写功能极其强大。

  • 完美兼容 .yaml 配置，无需转换
  • 极其精美的仪表盘 (Dashboard)
  • 强大的脚本覆写与去广告功能
  • UI 设计极具果粉美感
美区 App Store

终极选购建议

根据你的需求精准选择：

  • 追求高颜值、高性能：首选 Clash Verge Rev (Win) 或 Hiddify (Android)。
  • 追求极致稳定、低电耗：首选 v2rayN (Win) 或 v2rayNG (Android)。
  • 追求跨平台一致体验：首选 Karing。
  • iOS 用户：小火箭 是 Plan B 必留，Stash 是主力进阶。
